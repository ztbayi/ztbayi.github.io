如何创建一个博客

1.安装node.js。

node -v npm -v验证 (Hexo：搭建博客框架（wordpress，jekyllrb）)

2.安装cnpm。

npm install -g cnpm --registry=https://registry.npm.taobao.org

cnpm -v

3.安装hexo。

cnpm install -g hexo-cli

hexo-v验证

4.使用hexo搭建博客。

chdir

mkdir blog

cd blog

hexo init

5.创建博客

hexo new “我的第一篇博客”

hexo clean

hexo generate

hexo start

6.如何将自己的博客部署到github上公开使用

github里创建一个repository，并按照如下方式命名

回到cmd命令窗口，输入:

cnpm install --save hexo-deployer-git

下拉到最后,输入如上图所示配置，其中仓库地址为这个图里的，保存退出。

部署到远端，输入：

hexo deploy
