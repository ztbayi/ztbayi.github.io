创建一个博客

1.安装node.js。

node -v npm -v验证 (Hexo：搭建博客框架（wordpress，jekyllrb）)

2.安装cnpm。

npm install -g cnpm --registry=https://registry.npm.taobao.org

npm国内下载速度不太行，可选用淘宝镜像

cnpm -v

3.安装hexo。

cnpm install -g hexo-cli

hexo-v验证

4.使用hexo搭建博客。

chdir 查看路径

mkdir blog 创建文件夹

cd blog 位于blog路径下

hexo init 利用hexo在blog路径下生成博客

5.创建博客

hexo new “我的第一篇博客” 创建博客

hexo clean

hexo generate Generate static files

hexo start 在网页上查看博客
